Проект для предмета "Основы ИС" в университете. 

Тема для проекта была выбрана "Аэропорт". 

О проекте "Аэропорт":

Система содержит информацию об авиарейсах: номер рейса, пункт назначения,  пункты посадки,  дни и время вылета, стоимость билета и т.д.
Система должна обеспечивать: выдачу  списка  авиарейсов, пополнение списка; поиск информации о рейсе по одному из реквизитов.

Система должна предоставлять следующие функции:
1) Вывод списка авиарейсов — отображение всех зарегистрированных рейсов с основной информацией (номер рейса, пункт назначения, пункты посадки, дни и время вылета, стоимость билета).
2) Добавление нового рейса — возможность пополнения данных новыми рейсами.
3) Поиск рейса — поиск и отображение информации по номеру рейса, пункту назначения.
4) Удаление данных (по необходимости).

Каждый рейс должен содержать следующие поля:

1) Номер рейса (уникальный идентификатор);
2) Пункт назначения;
3) Пункты посадки;
4) Дни вылета (например, Пн, Ср, Пт);
5) Время вылета;
6) Стоимость билета;
